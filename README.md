# **SNDrawerViewController**

SNDrawerViewController 支持左抽屉，支持右抽屉，支持左右双抽屉。并且可以定制抽屉大小以及主界面的尺寸，因为它只是一个抽屉控制器框架没有任何的子视图，所以它的外观完全需要定制。
